Linux基础命令学习

Aug 30, 2017 | Linux | 阅读

浪沙沙同学想学习Linux的命令，便写下这个简单的关于Linux命令学习教程

Linux基础命令包括

  1. ls
  2. man
  3. chmod
  4. vim

ls命令

ls命令用来列出当前目录下的内容
ls的参数常用的包括

  1. ls -a (Include directory entries whose names begin with a dot (.) 将隐藏的内容展示出来)
  2. ls -l （Force output to be one entry per line.）
  3. ls -alh (将目录下全部内容包括隐藏内容，按一行的形式输出，并且文件大小以常用字节形式输出)

man命令

man命令是manual的缩写，是系统中提供相关条目参考信息的命令。
要想找某一个命令的详细信息，比如ls命令，只要在命令行输入man后跟随空格及ls．然后回车就可以了。
man 命令是作为初学者学习命令的非常好的工具，在遇到自己不懂的命令时，只需要man一下相关的命令，就可以获得该指令的详细信息内容。进入man命令后，按q就可以退出。按↑↓键进行翻页。

chmod命令

使用命令chmod来为文件或目录赋予权限的。在UNIX系统家族里，文件或目录权限的控制分别以读取 r ，写入 w ，执行 x 3种一般权限来区分，它们代表的数字分别为4，2，1。以及每3组字母或者-代表不同的用户范围

  1. 第一组 表示“用户（user）”，即文件或目录的所有者。
  2. 第二组 表示“同组（group）用户”，即与文件属主有相同组ID的所有用户。
  3. 第三组 表示“其他（others）用户”。

vim命令

vim作为Linux下的文本编译器命令
vim file
vim有两个模式
正常模式（按Esc） 左下角显示文件名或为空 在该模式下，可以查看文本，输入退出命令 q，以及退出保存命令 wq
插入模式（按i键进入） 左下角显示 –INSERT– 在该模式下可以进行输入内容
