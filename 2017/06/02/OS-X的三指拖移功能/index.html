OS X的三指拖移功能

Jun 2, 2017 | OS X | 阅读

自从OS X更新EI Capitan 起，macOS 上针对触控板的「三指拖移」功能被隐藏。需要额外的方式启动。
打开「系统偏好设置」，点选「辅助功能」；
点选「鼠标与触控板」，打开「触控板选项」；
勾选「启用拖移」，点选「三指拖移」，保存退出。
